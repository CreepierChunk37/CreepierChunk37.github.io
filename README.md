# Academic Homepage

Academic homepage for `CreepierChunk37.github.io`, based on the CC0-licensed [Minimal Light](https://github.com/yaoyao-liu/minimal-light) Jekyll theme.

## Personalize

- Edit `_config.yml` for your name, affiliation, email, avatar, and profile links.
- Edit `index.md` for your biography, research interests, publications, projects, and education.

Push the site to the `main` branch of a public repository named `CreepierChunk37.github.io`. GitHub Pages will publish it at <https://creepierchunk37.github.io/>.
